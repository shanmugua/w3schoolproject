JS Variables

There are 4 ways to declare a JavaScript variable: using var, using let, using const, or using nothing.

Variable Declarations
JavaScript
Try it »
// Using let (block-scoped, reassignable)
let name = "Alice";
name = "Bob";  // ✓ OK

// Using const (block-scoped, not reassignable)
const pi = 3.14159;
// pi = 3; ✗ TypeError!

// Avoid var in modern JS
var age = 25;

JS Functions

A JavaScript function is a block of code designed to perform a particular task. A function is executed when "something" invokes (calls) it.

Function Syntax
JavaScript
Try it »
// Traditional function
function greet(name) {
  return "Hello, " + name + "!";
}

// Arrow function (ES6+)
const greet = (name) => `Hello, ${name}!`;

console.log(greet("Alice"));  // "Hello, Alice!"

JS Arrays

An array is a special variable, which can hold more than one value:

Array Methods
JavaScript
Try it »
const fruits = ["Apple", "Banana", "Cherry"];

// Access elements
console.log(fruits[0]);  // "Apple"

// Array methods
fruits.push("Mango");     // Add to end
fruits.pop();              // Remove from end
fruits.map(f => f.toUpperCase());
fruits.filter(f => f.startsWith("A"));
fruits.find(f => f === "Cherry");

JS DOM

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

DOM Manipulation
JavaScript
Try it »
// Select elements
const el = document.querySelector("#myId");
const els = document.querySelectorAll(".myClass");

// Modify content
el.innerHTML = "New content";
el.style.color = "red";
el.classList.add("active");

// Events
el.addEventListener("click", () => {
  alert("Clicked!");
});

JS Async/Await

Async/await makes asynchronous code look and behave a little more like synchronous code.

Fetch API with Async/Await
JavaScript
async function fetchData() {
  try {
    const response = await fetch("https://api.example.com/data");
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error("Error:", error);
  }
}
